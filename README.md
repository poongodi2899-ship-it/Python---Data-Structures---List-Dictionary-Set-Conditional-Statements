# 📊 Python – Data Structures & Conditional Statements

## 📝 Project Overview
This assignment focuses on understanding Python's core data structures and conditional statements. It demonstrates practical implementation of Lists, Dictionaries, Sets, and decision-making using `if`, `elif`, and `else` statements.

---

## 🎯 Objective
- Learn Python List operations
- Understand Dictionary creation and manipulation
- Explore Set properties and operations
- Implement Conditional Statements for decision-making
- Improve problem-solving and programming skills

## 🚀 Key Features

### 📋 List Operations
- Created and modified lists
- Added, inserted, removed, and sorted elements
- Calculated maximum, minimum, and total values
- Accessed list elements using indexing and slicing

### 📖 Dictionary Operations
- Created a student marks dictionary
- Added and updated student records
- Retrieved keys, values, and key-value pairs

### 🔄 Set Operations
- Created sets with unique values
- Performed Union and Intersection operations
- Understood duplicate removal and indexing limitations

### ⚡ Conditional Statements

- Classified performance using `if`, `elif`, and `else`
- Displayed personalized feedback messages
